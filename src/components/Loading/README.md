## Loading

```javascript
// 引入
import Loading from '~/components/Loading';

// 设置(非必要，不设置使用默认样式)
Loading.spinBgColor = 'rgba(255, 255, 0, 0.2)'; // loading图标的背景
Loading.spinColor = 'green'; // loading图标的颜色
Loading.spinHeight = '0.4rem'; // loading图标，尺寸。

// 调用
Loading.show(); // 显示
Loading.hide(); // 隐藏
```
